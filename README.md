### Engineering Philosophy: Security-First
I treat production infrastructure like financial governance: it must be resilient, scalable, and designed to prevent cascading failures. My transition from Finance to DevOps allows me to view second-order effects through both a fiscal and a technical lens.

* **Security-First Mandate:** In fintech, security isn't a feature; it's the foundation. I architect systems using Zero-Trust principles, ensuring that every data packet and API call is authenticated, authorized, and encrypted.
* **Fiscal-Technical Convergence:** I build infrastructure that respects the "unit economics" of the code—minimizing latency while maximizing auditability.
